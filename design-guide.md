# 디자인 가이드

---

## 1. 프로젝트 개요

**캠페인명**: 케다부커클럽 | kakao!mpact
**주제**: 이동약자를 위한 AI 모델 만들기
**목표**: 실내 접근성 데이터 수집 캠페인을 통한 누구나 이동하기 쉬운 세상 구현

## 3. 타이포그래피

### 3.1 헤드라인

- **Font Family**: 산돌 고딕 Neo (Sandoll Gothic Neo) 또는 유사 한글 폰트
- **Font Weight**: Extra Bold (800-900)
- **Font Size**:
  - 메인 헤드라인: 48-64px
  - 서브 헤드라인: 32-40px
- **Line Height**: 1.2-1.4
- **Color**:
  - White (`#FFFFFF`) - 컬러 배경 위
  - Black (`#000000`) - 라이트 배경 위

### 3.2 본문 텍스트

- **Font Weight**: Regular (400), Medium (500)
- **Font Size**:
  - 본문: 16-18px
  - 캡션: 14-15px
- **Line Height**: 1.6-1.8
- **Color**: `#333333` ~ `#4A4A4A`

### 3.3 강조 텍스트

- **Highlight Color**: `#4A90FF` (파란색)
- **사용 예시**: "누군가의 '이동'"

## 4. 레이아웃 시스템

### 4.1 그리드 시스템

- **컨테이너 최대 너비**: 1200px (데스크톱)
- **여백 (Padding)**:
  - 모바일: 20-24px
  - 태블릿: 40px
  - 데스크톱: 60-80px
- **섹션 간격**: 80-120px

### 4.2 카드 레이아웃

- **Border Radius**: 16-20px
- **Shadow**: `0px 4px 20px rgba(0, 0, 0, 0.08)`
- **Padding**: 24-32px
- **배경색**: White (`#FFFFFF`)

---

## 5. 일러스트레이션 스타일

### 5.1 아이콘 스타일

- **스타일**: 플랫 디자인, 미니멀
- **색상**:
  - Primary: `#FF6B00` (오렌지)
  - Secondary: `#4A4A4A` (다크 그레이)
  - Background: `#FFE8C2` (크림)
- **사이즈**:
  - 소형: 48x48px
  - 중형: 96x96px
  - 대형: 180x180px

### 5.2 주요 모티프

1. **휠체어** - 이동약자를 상징
2. **망치** - 문제 해결/개선을 상징
3. **카메라** - 사진 촬영/데이터 수집을 상징

### 5.3 일러스트레이션 원칙

- 심플하고 직관적인 형태
- 2-3가지 컬러로 제한
- 그림자와 그라데이션 최소화
- 친근하고 접근하기 쉬운 느낌

## 6. UI 컴포넌트

### 6.1 버튼

#### Primary Button
```
배경색: #FF6B00
텍스트 색상: #FFFFFF
Font Weight: Bold (700)
Border Radius: 8px
Padding: 14px 32px
Font Size: 16px
상태:
  - Hover: #FF5500
  - Active: #E65500
```

#### Secondary Button
```
배경색: #FFFFFF
텍스트 색상: #FF6B00
Border: 2px solid #FF6B00
Border Radius: 8px
Padding: 14px 32px
Font Size: 16px
```

### 6.2 정보 박스

**3단 구성 카드**:
- **레이아웃**: 수평 3열 그리드
- **배경색**:
  - 옵션 1: `#FFF5EB` (연한 오렌지)
  - 옵션 2: `#F8F8F8` (연한 회색)
- **Border**: None 또는 `1px solid #EEEEEE`
- **Border Radius**: 12px
- **Padding**: 24px
- **간격**: 16-20px

**카드 내용 구조**:
- 아이콘/이모지 (상단)
- 제목 (Bold, 16-18px)
- 설명 텍스트 (Regular, 14-16px)

### 6.3 모바일 화면 디자인

#### 상태바
- 시간, 신호, 배터리 표시
- 높이: 44px (iOS 기준)

#### 카드 UI
- **Border Radius**: 24px (모바일에서 더 둥글게)
- **Padding**: 32px
- **이미지 Border Radius**: 16px

#### CTA 버튼
- **배경색**: Black (`#000000`)
- **텍스트**: White, Uppercase
- **Border Radius**: 24px
- **위치**: 중앙 하단

## 7. 이미지 가이드

### 7.1 사진 스타일

- **장소 사진**: 실내 공간 (카페, 로비 등)
- **구도**: 클린하고 밝은 느낌
- **보정**: 자연스러운 색감, 높은 명도

### 7.2 이미지 비율

- **카드 썸네일**: 16:9 또는 4:3
- **Border Radius**: 12-16px
- **갤러리 그리드**: 2열 (모바일), 3-4열 (데스크톱)

---

## 8. 인터랙션 & 애니메이션

### 8.1 버튼 인터랙션

- **Hover**:
  - 배경색 변화 (0.2s ease)
  - Scale: 1.02
- **Active**:
  - Scale: 0.98
  - 배경색 더 진하게

### 8.2 카드 인터랙션

- **Hover**:
  - Shadow 증가: `0px 8px 30px rgba(0, 0, 0, 0.12)`
  - Transform: `translateY(-4px)`
  - Transition: 0.3s ease

## 9. 접근성 (Accessibility)

### 9.1 컬러 대비

- **텍스트와 배경 대비비**: 최소 4.5:1 (WCAG AA)
- **주요 CTA**: 최소 7:1 (WCAG AAA)

### 9.2 터치 타겟

- **최소 크기**: 44x44px (모바일)
- **간격**: 최소 8px

### 9.3 폰트 크기

- **최소 본문 크기**: 16px (모바일)
- **가독성**: 명조체 대신 고딕체 사용

## 12. 콘텐츠 섹션별 가이드

### 12.1 히어로 섹션 (페이지 1)

- **배경**: 그라데이션 오렌지 (`#FF9D42` → `#FFA850`)
- **레이아웃**: 좌측 텍스트, 우측 일러스트
- **텍스트 정렬**: 좌측 정렬
- **일러스트**: 3개의 카드 형태, 회전된 배치

### 12.2 캠페인 소개 섹션 (페이지 2)

- **배경**: White
- **제목**: 큰 사이즈, 블루 강조
- **3단 정보 박스**: 크림 배경
- **모바일 목업**: 센터 배치, 그림자 효과

### 12.3 기술 설명 섹션 (페이지 3)

- **배경**: 라이트 블루
- **레이아웃**: 텍스트 + 데스크톱 목업
- **3단 기술 카드**: 화이트 배경, 아이콘 + 텍스트

## 15. 디자인 원칙

1. **접근성 우선**: 모든 사용자가 쉽게 이해할 수 있는 디자인
2. **명확한 계층**: 정보의 우선순위가 명확하게 전달
3. **따뜻한 느낌**: 사회적 가치를 전달하는 친근한 톤
4. **심플함**: 불필요한 장식 제거, 본질에 집중
5. **일관성**: 전체 페이지에서 동일한 디자인 언어 사용
