# 동접랩 성과 공유 패널

2-Line Pipeline Panel - 실내 접근성 데이터 처리 파이프라인 시각화

## 프로젝트 구조

```
panel-test/
├── index.html      # HTML 구조 (메인 페이지)
├── styles.css      # 스타일시트 (디자인 & 레이아웃)
├── app.js          # 애플리케이션 로직 (이벤트 처리 & 애니메이션)
├── data.json       # 콘텐츠 데이터 (노드 정보 & 설명)
└── README.md       # 프로젝트 문서
```

## 파일 설명

### `index.html`
- 페이지의 HTML 구조를 정의합니다
- 헤더, 파이프라인 캔버스, 상세 정보 패널, 푸터로 구성
- 시맨틱 HTML과 접근성 속성을 사용합니다

### `styles.css`
- 전체 페이지의 스타일과 레이아웃을 관리합니다
- CSS 변수를 사용하여 색상과 크기를 중앙에서 관리
- 반응형 디자인 지원 (1250px 브레이크포인트)
- 다크 테마 및 그라데이션 배경

### `app.js`
- 모든 인터랙티브 기능을 처리합니다
- 노드 클릭, 키보드 내비게이션, 자동 재생
- SVG 커넥터 동적 생성
- 애니메이션 플로우 도트 관리
- `data.json`에서 콘텐츠를 동적으로 로드

### `data.json`
- 파이프라인 노드의 모든 콘텐츠를 저장합니다
- 각 노드의 제목, 설명, 어필 문구, 불렛 포인트 포함
- 노드 순서 정의
- 콘텐츠 수정 시 이 파일만 편집하면 됩니다

## 사용 방법

### 로컬에서 실행

웹 서버를 통해 실행해야 합니다 (CORS 제한으로 인해 파일을 직접 열면 작동하지 않습니다):

```bash
# Python 3 사용
python3 -m http.server 8000

# 또는 Node.js의 http-server 사용
npx http-server -p 8000
```

그런 다음 브라우저에서 `http://localhost:8000` 접속

### 키보드 단축키

- `←/→` (좌우 화살표): 이전/다음 단계로 이동
- `Space`: 자동 재생 토글 (ON/OFF)
- `Enter`: 포커스된 노드 선택

## 콘텐츠 수정

콘텐츠를 수정하려면 `data.json` 파일만 편집하면 됩니다:

```json
{
  "order": [...],  // 노드 표시 순서
  "nodes": {
    "node_id": {
      "head": "노드 제목",
      "desc": "노드 설명",
      "appealTitle": "어필 제목 (HTML 지원)",
      "appealSub": "어필 부제목",
      "bullets": ["포인트 1", "포인트 2", ...]
    }
  }
}
```

## 스타일 커스터마이징

`styles.css` 파일의 `:root` 섹션에서 CSS 변수를 수정하여 색상과 크기를 조정할 수 있습니다:

```css
:root {
  --accent: #7C5CFF;     /* 주요 강조 색상 */
  --accent2: #2EE59D;    /* 보조 강조 색상 */
  --bg0: #070A12;        /* 배경 색상 1 */
  --bg1: #0B1020;        /* 배경 색상 2 */
  /* ... */
}
```

## 기능

### 파이프라인 시각화
- 2개의 레인으로 구성된 파이프라인
- 상단: 모델 구축 과정 (데이터 수집 → 필터링 → 라벨링 → 모델 구축)
- 하단: 모델 적용 과정 (이미지 확보 → 자동 분석 → 검수)

### 인터랙션
- 각 노드 클릭 시 상세 정보 표시
- 자동 재생 모드 (6.5초마다 다음 단계로 이동)
- 플로우 애니메이션 (파이프라인을 따라 이동하는 도트)
- SVG 커넥터로 병렬 수집 노드를 병합 노드로 연결

### 반응형 디자인
- 데스크톱: 2열 레이아웃 (파이프라인 + 상세 정보)
- 모바일: 1열 스택 레이아웃
- 접근성: 키보드 내비게이션, ARIA 레이블, 시맨틱 HTML

## 유지보수 가이드

### 새 노드 추가
1. `data.json`의 `nodes` 객체에 새 노드 데이터 추가
2. `data.json`의 `order` 배열에 노드 ID 추가
3. `index.html`에 해당 노드의 HTML 마크업 추가

### 스타일 변경
- 전역 스타일: `styles.css`의 `:root` 섹션 수정
- 컴포넌트별 스타일: 해당 클래스 선택자 찾아서 수정

### 애니메이션 조정
- `app.js`의 `startAuto()`: 자동 재생 간격 변경 (현재 6500ms)
- `app.js`의 `startFlow()`: 플로우 도트 속도 변경 (현재 60ms 간격, 0.075 증가량)
